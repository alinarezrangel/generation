What is Generation

Generation is a CSS3 based framework for web pages, with support of HTML5 and ECMAScript 5 or 6.

Generation provides an easy interface for build web pages only using CSS3 and, if you need it, ECMAScript 5 or 6.

What is the difference between Generation and other CSS-based framework?

Generation introduces a builtins support for CSS3 features as flexboxes, media queries, animations, etc.

What are Generation color palletes

A color pallete is an external CSS file with aditional colors for general purposes.

What is a Generation JS Plugin

Generation includes a minimal ECMAScript (with support for JavaScript) 5 or 6 core, and you can write or download plugins for that JS core.
