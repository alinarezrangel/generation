Generation

Generation is a CSS3 framework which uses the best from modern web technologies to build responsive, mobile-first web pages and applications.

Simple and responsive

Generation is simple to use and responsive without any change, it makes use of flexbox layout which enables the content to move and flex around the page when it is resized without using media queries.

Extendible and colorful

Generation uses SASS and have well-documented classes, mixins and functions. You can extend Generation to create new elements, widgets and your very own color paletes.

Semantic

No more content classes, write the HTML that you just need and no more. Generation reinforces the semantic web design with powerful, semantic and general classes.

Newer and better: is Generation 2

Generation 2 is the new iteration of a complete and flexible framework, now redesigned from scratch using SASS as the core language and with SASS comes a new powerful API for color paletes, classes and elements.
