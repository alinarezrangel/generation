Text: font and sizes

Generation 1 have a collection of classes for styling text. The basic classes are for style the text size and text font family.

The Generation 1 supported classes for font sizes are:

  • text-tiny: A very small text (~0.7em)
  • text-small: A small text (~0.9em)
  • text-normal: Default font size (~1em)
  • text-big: A big text (~1.2em)
  • text-ultra-big: A very big text (~1.5em)
  • text-large: A very very big text (~1.9em)
  • text-xlarge: A very very very big text (~2em)
  • text-jumbo or text-jumbo-1: A jumbo text (~3.2em)
  • text-jumbo-2: A very jumbo text (~3.2)
  • text-jumbo-3: A very very jumbo text (~3.7em)
  • text-jumbo-4: A big jumbo text (~4.2em)
  • text-jumbo-5: A very big jumbo text (~5em)

For example: in this page, the first title ("Generation", in the top of the page) is a text-jumbo-5, but in the page footer, the text inside the columns is a text-small.

Generation provides a form for set font style and family:

  • font-normal: The default font, Arial sans-serif.
  • font-monospace: A monospaced font.
  • font-serif: The "Times New Roman" font, serif.
  • font-bold: A bold font.
  • font-italic: A italic font.

You can use this and colors for styling text.
