Colors: Getting Started

In Generation, all colors have a intuitive syntax:

color-name
text-color-name
border-color-name

Where name is:

light-cname
dark-cname
cname

And cname is a color name. The color names are defined in color palletes, you can use a color pallete, then you need read the documentation for that color pallete for get a list with the defined color names. In Generation 1, the predefined color names are:

The syntax color- sets the background color, text-color- sets the text color and border-color- sets the border color.

Warning!

The colors black and white not have a dark and light versions, try with grey color.
