Layouts: Blocks, Flexibles and boxes

In Generation, the layout system is divided into three main groups:

Blocks

A block is a simple element with fixed width1 but variable height. These elements have properties like float, position and margin, and they are the most simple layout system.

The above element is a flexible, you can see this viewing the page source code.

The main difference between a block and a flexible is how a block and a flexible structures it's contents: A block displays it's elements, but a flexible layouts its with a flexible layout model where each item in it is rendered with a position, width and height relative to others childrens and relative to the main flexible container.

This not mean "where you see a centered text, is a flexible", a block display it's childrens using the CSS block model (see the CSS3 draft for more info).

Generation uses the flexible block model rather than the block model, because this flexible model is responsive, it displays better and all screens and have multiples properties than the block model.

Flexibles

Boxes

A box is like a block, but without margin, padding and any effect. The boxes are general purpose containers for any use (but if you need a container for text, elements, widgets or others, see the container class).
