Using Generation 1

Generation 1 is very easy to use, you only need two links: the CSS and the ECMAScript/JavaScript Core.

If you want it, you can not use the ECMAScript/JavaScript core, but then you can not use modals.

The two links look like these:

<link href="generation/css/generation.css" rel="stylesheet"/>
<script type="text/javascript" src="generation/js/generation.js"></script>

After this, you are ready to use Generation 1.

Warning!

If you not have Generation 1 installed in your page's root directory, then you need to install it or use a external link:

<link href="alinarezrangel.github.io/generation/css/generation.css" rel="stylesheet"/>
<script type="text/javascript" src="alinarezrangel.github.io/generation/js/generation.js"></script>

If you have Generation 1 installed, you can open a demonstration opening the file "index.html" in the Generation 1 directory.
